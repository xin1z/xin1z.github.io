This site was built mainly because I *accidentally* bought a domain at midnight and wanted to use it to do something *fun*. And due to my lack of knowledge on HTML, CSS and Javascript, I spent an afternoon (and the evening follows) learning some basic stuffs of HTML and CSS, then this site was built.

This site uses [Jekyll](https://github.com/jekyll/jekyll) to compile **Markdown files** into **HTML**. The theme used here is a modified version of [Minima](https://github.com/minima), which I've changed it a little bit with my poor HTML and CSS knowledge (don't laugh).

This site is hosted on [GitHub](https://github.com) with its *GitHub Page* feature, which means its static. Maybe I'll try something much more *dynamic*, but not now.
